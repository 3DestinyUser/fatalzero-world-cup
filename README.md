# FATALZERO World Cup

Prototipo educativo jugable para la campaña inicial de Trinca y Destrinca Segura.

VI3W+ by 3Destiny acompaña la experiencia como MICROcomunicador 3D: reconoce
el mapa, las fases de misión, las nueve dimensiones y la World Cup; resume el
contexto, registra señales del día y comunica desafíos y logros.

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
```

El sitio se publica automaticamente en GitHub Pages al actualizar la rama `main`.

## Integración VI3W+

El contrato `window.VI3W` para enviar notificaciones, contextos o logros desde
otros sistemas está documentado en [`docs/vi3w-world-cup.md`](docs/vi3w-world-cup.md).
